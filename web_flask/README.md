# AirBnB Clone - Web Framework

Flask web application routes for the AirBnB clone project.

## Tasks

- **0-hello_route.py**: `/` → "Hello HBNB!"
- **1-hbnb_route.py**: adds `/hbnb` → "HBNB"
- **2-c_route.py**: adds `/c/<text>`
- **3-python_route.py**: adds `/python/<text>` (default: "is cool")
- **4-number_route.py**: adds `/number/<int:n>`
- **5-number_template.py**: adds `/number_template/<int:n>` (HTML)
- **6-number_odd_or_even.py**: adds `/number_odd_or_even/<int:n>` (HTML)
- **7-states_list.py**: `/states_list` — list all States from storage
- **8-cities_by_states.py**: `/cities_by_states` — States with nested Cities
- **9-states.py**: `/states` and `/states/<id>`
- **10-hbnb_filters.py**: `/hbnb_filters` — filter page with States and Amenities

## Usage

```bash
python3 -m web_flask.0-hello_route
```
